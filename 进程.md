操作系统不信任任何用户---->给用户提供各种**系统调用接口**(接口实际上就是C的函数)

##### **进程大致了解：**

* 加载到内存的程序，就叫做进程

* 任何进程在形成时，操作系统要为该进程创建**PCB，进程控制块**

  **PCB---->struct task_struct{}**

  * ```c++
    //简单理解就是
    struct PCB{
        //进程的所有的属性
    }
    ```

  * 我们启动程序的过程本质都是在系统上面创建进程！！(查看进程：**ps axj |grep "test"**)(详细：**ps axj |head -1 && ps axj |grep "test"**)
